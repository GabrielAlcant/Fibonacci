# Fibonacci

Este repositório contém um script simples para calcular números da sequência de Fibonacci.

## Pré-requisitos

- Python 3.8 ou superior instalado.

## Como rodar

1. Abra um terminal e navegue até a pasta do projeto:
   ```bash
   cd /workspaces/Fibonacci
   ```
2. Execute o script passando um número inteiro como argumento para calcular o n-ésimo termo:
   ```bash
   python3 fibonacci.py 10
   ```
   Isso irá imprimir o décimo número da sequência de Fibonacci.

3. Alternativamente, você pode executar sem argumentos e seguir as instruções interativas (se suportado pelo script).

## Exemplo

```bash
$ python3 fibonacci.py 5
5
```

## Estrutura

- `fibonacci.py` - script principal que calcula o termo da sequência.
